|------------------|----------------------|-----------------|----------|---------|

| **Total**        |                      |                 |          | **$19,335** |

### **Paso 2: Estrategia de Financiación**
- **Plataforma**: Affirm + Splitit (Combinación)
- **Mecánica**:
  1. Comprar GPUs via Affirm (36 meses al 0% APR) → $1,799/cuota $50
  2. Adquirir resto de componentes con Splitit (24 cuotas) → $3,144/cuota $131

### **Paso 3: Flujo de Pago**
1. Generar tarjeta virtual Davivienda Global (Límite: $20,000 USD)
2. Vincular a Splitit/Affirm como método de pago
3. Distribuir compras:
   - **Newegg**: GPUs + PSUs (Affirm)
   - **B&H Photo**: Motherboards (Splitit)

### **Paso 4: Logística Internacional**
- **Casillero**: Aeropost (Tarifa Gold Tier)
- **Costo envío**: $1.2/lb → Peso total: 210 lbs → $252
- **Seguros**: 3% valor total → $580

**Costo Total Financiado**: $19,335 + $832 = $20,167 USD  
**Cuota Mensual**: $181 USD (Sin impacto cupo físico)
